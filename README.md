# 🏛️ 아타나시우스 부활절 색인 (Festal Index) 프로젝트

이 저장소는 4세기 알렉산드리아의 주교 아타나시우스(Athanasius)의 부활절 서신 발송 기록과 당시 이집트의 정치적/교회사적 사건을 정리한 디지털 아카이브입니다.

## 📌 프로젝트 개요
* **사료적 가치**: 4세기 이집트와 아타나시우스의 생애를 이해하는 데 핵심적인 가치를 지닌 문헌입니다.
* **언어**: 그리스어로 작성되었으나 현재는 시리아어로 보존되어 있습니다.
* **주요 내용**: 매년 부활절 날짜, 로마 집정관, 이집트 총독 명단 및 주요 사건 기록.
