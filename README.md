* [cite_start]**언어 및 전승**: 원래 그리스어로 작성되었으나 현재는 시리아어 필사본으로 보존되어 있습니다. [cite: 3]
* [cite_start]**핵심 기록**: 매년 부활절 날짜, 로마 집정관, 이집트 총독 명단 및 주요 사건 요약. [cite: 5, 6]
